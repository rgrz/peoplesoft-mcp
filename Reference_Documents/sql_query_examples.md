# PeopleSoft SQL Query Examples

Real-world query patterns for common business scenarios. All queries use proper effective-dating.

## Employee Data Queries

### Get Current Job Record for an Employee

The foundational pattern for getting the most current job data.

```sql
SELECT *
FROM PS_JOB A
WHERE A.EMPLID = :emplid
  AND A.EMPL_RCD = (
      SELECT MAX(A1.EMPL_RCD) 
      FROM PS_JOB A1 
      WHERE A1.EMPLID = A.EMPLID
  )
  AND A.EFFDT = (
      SELECT MAX(A2.EFFDT) 
      FROM PS_JOB A2 
      WHERE A2.EMPLID = A.EMPLID 
        AND A2.EMPL_RCD = A.EMPL_RCD
        AND A2.EFFDT <= SYSDATE
  )
  AND A.EFFSEQ = (
      SELECT MAX(A3.EFFSEQ) 
      FROM PS_JOB A3 
      WHERE A3.EMPLID = A.EMPLID 
        AND A3.EMPL_RCD = A.EMPL_RCD 
        AND A3.EFFDT = A.EFFDT
  )
```

### Find All Active Employees

```sql
SELECT A.EMPLID, A.HR_STATUS, A.EMPL_STATUS
FROM PS_JOB A
WHERE A.EFFDT = (
    SELECT MAX(A1.EFFDT) 
    FROM PS_JOB A1 
    WHERE A1.EMPLID = A.EMPLID 
      AND A1.EMPL_RCD = A.EMPL_RCD
      AND A1.EFFDT <= SYSDATE
)
AND A.EFFSEQ = (
    SELECT MAX(A2.EFFSEQ) 
    FROM PS_JOB A2 
    WHERE A2.EMPLID = A.EMPLID 
      AND A2.EMPL_RCD = A.EMPL_RCD 
      AND A2.EFFDT = A.EFFDT
)
AND A.HR_STATUS = 'A'
```

### Employee with Department and Location Names

```sql
SELECT
    A.EMPLID,
    A.EMPL_RCD,
    B.NAME,
    A.DEPTID,
    C.DESCR AS DEPARTMENT_NAME,
    A.LOCATION,
    D.DESCR AS LOCATION_NAME
FROM PS_JOB A
INNER JOIN PS_PERSONAL_DATA B 
    ON A.EMPLID = B.EMPLID
INNER JOIN PS_DEPT_TBL C 
    ON A.SETID_DEPT = C.SETID 
    AND A.DEPTID = C.DEPTID
    AND C.EFFDT = (
        SELECT MAX(C1.EFFDT) 
        FROM PS_DEPT_TBL C1 
        WHERE C1.SETID = C.SETID 
          AND C1.DEPTID = C.DEPTID 
          AND C1.EFFDT <= SYSDATE
    )
INNER JOIN PS_LOCATION_TBL D 
    ON A.SETID_LOCATION = D.SETID
    AND A.LOCATION = D.LOCATION
    AND D.EFFDT = (
        SELECT MAX(D1.EFFDT) 
        FROM PS_LOCATION_TBL D1 
        WHERE D1.SETID = D.SETID
          AND D1.LOCATION = D.LOCATION 
          AND D1.EFFDT <= SYSDATE
    )
WHERE A.EFFDT = (
    SELECT MAX(A1.EFFDT) 
    FROM PS_JOB A1 
    WHERE A1.EMPLID = A.EMPLID 
      AND A1.EMPL_RCD = A.EMPL_RCD 
      AND A1.EFFDT <= SYSDATE
)
AND A.EFFSEQ = (
    SELECT MAX(A2.EFFSEQ) 
    FROM PS_JOB A2 
    WHERE A2.EMPLID = A.EMPLID 
      AND A2.EMPL_RCD = A.EMPL_RCD 
      AND A2.EFFDT = A.EFFDT
)
AND A.HR_STATUS = 'A'
```

### Search Employee by Name (Case-Insensitive)

```sql
SELECT
    A.EMPLID,
    A.NAME,
    A.LAST_NAME,
    A.FIRST_NAME
FROM PS_PERSONAL_DATA A
WHERE UPPER(A.FIRST_NAME) LIKE UPPER(:first_name) || '%'
  AND UPPER(A.LAST_NAME) LIKE UPPER(:last_name) || '%'
```

## Hiring & Termination Queries

### New Hires in Date Range

```sql
SELECT
    A.EMPLID,
    B.NAME,
    A.HIRE_DT,
    A.LAST_HIRE_DT
FROM PS_EMPLOYMENT A
INNER JOIN PS_PERSONAL_DATA B 
    ON A.EMPLID = B.EMPLID
WHERE A.HIRE_DT BETWEEN :start_date AND :end_date
ORDER BY A.HIRE_DT DESC
```

### Employees by Compensation in Department

```sql

    A.EMPLID,
    B.NAME,
    A.DEPTID,
    A.ANNUAL_RT,
    A.COMPRATE
FROM PS_JOB A
INNER JOIN PS_PERSONAL_DATA B 
    ON A.EMPLID = B.EMPLID
WHERE A.EFFDT = (
    SELECT MAX(A1.EFFDT) 
    FROM PS_JOB A1 
    WHERE A1.EMPLID = A.EMPLID 
      AND A1.EMPL_RCD = A.EMPL_RCD 
      AND A1.EFFDT <= SYSDATE
)
AND A.EFFSEQ = (
    SELECT MAX(A2.EFFSEQ) 
    FROM PS_JOB A2 
    WHERE A2.EMPLID = A.EMPLID 
      AND A2.EMPL_RCD = A.EMPL_RCD 
      AND A2.EFFDT = A.EFFDT
)
AND A.DEPTID = :deptid
AND A.HR_STATUS = 'A'
```

## Contract Queries

### Active Contracts for Employee

```sql
SELECT
    A.EMPLID,
    B.NAME,
    A.CONTRACT_NUM,
    A.CONTRACT_TYPE,
    A.CONTRACT_BEGIN_DT,
    A.CONTRACT_END_DT,
    A.CONTRACT_STATUS
FROM PS_CONTRACT A
INNER JOIN PS_PERSONAL_DATA B 
    ON A.EMPLID = B.EMPLID
WHERE A.EMPLID = :emplid
  AND A.EFFDT = (
      SELECT MAX(A1.EFFDT)
      FROM PS_CONTRACT A1
      WHERE A1.EMPLID = A.EMPLID
        AND A1.EMPL_RCD = A.EMPL_RCD
        AND A1.CONTRACT_NUM = A.CONTRACT_NUM
        AND A1.EFFDT <= SYSDATE
  )
  AND A.CONTRACT_STATUS = 'A'
  AND (A.CONTRACT_END_DT >= SYSDATE OR A.CONTRACT_END_DT IS NULL)
```

### Contracts Expiring in Next 30 Days

```sql
SELECT
    A.EMPLID,
    B.NAME,
    A.CONTRACT_NUM,
    A.CONTRACT_TYPE,
    A.CONTRACT_BEGIN_DT,
    A.CONTRACT_END_DT
FROM PS_CONTRACT A
INNER JOIN PS_PERSONAL_DATA B 
    ON A.EMPLID = B.EMPLID
WHERE A.EFFDT = (
    SELECT MAX(A1.EFFDT)
    FROM PS_CONTRACT A1
    WHERE A1.EMPLID = A.EMPLID
      AND A1.EMPL_RCD = A.EMPL_RCD
      AND A1.CONTRACT_NUM = A.CONTRACT_NUM
      AND A1.EFFDT <= SYSDATE
)
AND A.CONTRACT_STATUS = 'A'
AND A.CONTRACT_END_DT IS NOT NULL
AND A.CONTRACT_END_DT BETWEEN SYSDATE AND SYSDATE + 30
ORDER BY A.CONTRACT_END_DT
```

## Organization Queries

### Active Departments

```sql
SELECT
    A.SETID,
    A.DEPTID,
    A.DESCR AS DEPARTMENT_NAME,
    A.EFF_STATUS
FROM PS_DEPT_TBL A
WHERE A.EFFDT = (
    SELECT MAX(A1.EFFDT)
    FROM PS_DEPT_TBL A1
    WHERE A1.SETID = A.SETID
      AND A1.DEPTID = A.DEPTID
      AND A1.EFFDT <= SYSDATE
)
AND A.EFF_STATUS = 'A'

```

### Headcount by Company, Location, Department

```sql
SELECT
    C.DESCR AS COMPANY_NAME,
    L.DESCR AS LOCATION_NAME,
    D.DESCR AS DEPARTMENT_NAME,
    COUNT(DISTINCT J.EMPLID) AS HEADCOUNT
FROM PS_JOB J
INNER JOIN PS_COMPANY_TBL C 
    ON J.COMPANY = C.COMPANY
    AND C.EFFDT = (
        SELECT MAX(C1.EFFDT)
        FROM PS_COMPANY_TBL C1
        WHERE C1.COMPANY = C.COMPANY
          AND C1.EFFDT <= SYSDATE
    )
INNER JOIN PS_LOCATION_TBL L 
    ON J.SETID_LOCATION = L.SETID
    AND J.LOCATION = L.LOCATION
    AND L.EFFDT = (
        SELECT MAX(L1.EFFDT)
        FROM PS_LOCATION_TBL L1
        WHERE L1.SETID = L.SETID
          AND L1.LOCATION = L.LOCATION
          AND L1.EFFDT <= SYSDATE
    )
INNER JOIN PS_DEPT_TBL D 
    ON J.SETID_DEPT = D.SETID
    AND J.DEPTID = D.DEPTID
    AND D.EFFDT = (
        SELECT MAX(D1.EFFDT)
        FROM PS_DEPT_TBL D1
        WHERE D1.SETID = D.SETID
          AND D1.DEPTID = D.DEPTID
          AND D1.EFFDT <= SYSDATE
    )
WHERE J.EFFDT = (
    SELECT MAX(J1.EFFDT)
    FROM PS_JOB J1
    WHERE J1.EMPLID = J.EMPLID
      AND J1.EMPL_RCD = J.EMPL_RCD
      AND J1.EFFDT <= SYSDATE
)
AND J.EFFSEQ = (
    SELECT MAX(J2.EFFSEQ)
    FROM PS_JOB J2
    WHERE J2.EMPLID = J.EMPLID
      AND J2.EMPL_RCD = J.EMPL_RCD
      AND J2.EFFDT = J.EFFDT
)
AND J.HR_STATUS = 'A'
GROUP BY C.DESCR, L.DESCR, D.DESCR
ORDER BY C.DESCR, L.DESCR, D.DESCR
```

## Manager/Reporting Queries

### Employee with Manager (Two-Level)

```sql
SELECT
    MGR.NAME AS MANAGER_NAME,
    MGR_JOB.EMPLID AS MANAGER_ID,
    MGR_JOB.JOBCODE AS MANAGER_JOB_CODE,
    EMP.NAME AS EMPLOYEE_NAME,
    EMP_JOB.EMPLID AS EMPLOYEE_ID,
    EMP_JOB.JOBCODE AS EMPLOYEE_JOB_CODE
FROM PS_JOB EMP_JOB
INNER JOIN PS_PERSONAL_DATA EMP 
    ON EMP_JOB.EMPLID = EMP.EMPLID
INNER JOIN PS_JOB MGR_JOB
    ON EMP_JOB.SUPERVISOR_ID = MGR_JOB.EMPLID
INNER JOIN PS_PERSONAL_DATA MGR 
    ON MGR_JOB.EMPLID = MGR.EMPLID
WHERE EMP_JOB.EFFDT = (
    SELECT MAX(E1.EFFDT)
    FROM PS_JOB E1
    WHERE E1.EMPLID = EMP_JOB.EMPLID
      AND E1.EMPL_RCD = EMP_JOB.EMPL_RCD
      AND E1.EFFDT <= SYSDATE
)
AND EMP_JOB.EFFSEQ = (
    SELECT MAX(E2.EFFSEQ)
    FROM PS_JOB E2
    WHERE E2.EMPLID = EMP_JOB.EMPLID
      AND E2.EMPL_RCD = EMP_JOB.EMPL_RCD
      AND E2.EFFDT = EMP_JOB.EFFDT
)
AND EMP_JOB.HR_STATUS = 'A'
AND EMP_JOB.SUPERVISOR_ID IS NOT NULL
AND MGR_JOB.EFFDT = (
    SELECT MAX(M1.EFFDT)
    FROM PS_JOB M1
    WHERE M1.EMPLID = MGR_JOB.EMPLID
      AND M1.EMPL_RCD = MGR_JOB.EMPL_RCD
      AND M1.EFFDT <= SYSDATE
)
AND MGR_JOB.EFFSEQ = (
    SELECT MAX(M2.EFFSEQ)
    FROM PS_JOB M2
    WHERE M2.EMPLID = MGR_JOB.EMPLID
      AND M2.EMPL_RCD = MGR_JOB.EMPL_RCD
      AND M2.EFFDT = MGR_JOB.EFFDT
)
AND MGR_JOB.HR_STATUS = 'A'
ORDER BY MGR.NAME, EMP.NAME
```

## Dependent Queries

### Employee Dependents with National ID

```sql
SELECT
    A.EMPLID,
    E.NAME AS EMPLOYEE_NAME,
    A.DEPENDENT_BENEF AS DEPENDENT_ID,
    A.NAME AS DEPENDENT_NAME,
    A.BIRTHDATE,
    A.SEX,
    A.RELATIONSHIP,
    A.DEP_BENEF_TYPE,
    B.NATIONAL_ID,
    B.NATIONAL_ID_TYPE,
    A.STUDENT,
    A.DISABLED
FROM PS_DEPENDENT_BENEF A
INNER JOIN PS_PERSONAL_DATA E 
    ON A.EMPLID = E.EMPLID
LEFT JOIN PS_DEP_BENEF_NID B 
    ON A.EMPLID = B.EMPLID 
    AND A.DEPENDENT_BENEF = B.DEPENDENT_BENEF
WHERE A.EMPLID = :emplid
ORDER BY A.NAME
```

## ePerformance Queries

### Employee Performance Documents

```sql
SELECT
    A.EP_APPR_ID,
    A.EMPLID,
    B.NAME,
    A.EP_REVIEWER_ID,
    A.PERIOD_BEGIN_DT,

    ON A.EMPLID = B.EMPLID

```

### Performance Ratings by Appraisal

```sql
SELECT
    A.EP_APPR_ID,
    A.EP_SECTION_ID,
    A.EP_ITEM_ID,
    A.EP_RATING,
    A.EP_WEIGHT,
    B.DESCRLONG AS ITEM_DESCRIPTION
FROM PS_EP_APPR_ITEM A
LEFT JOIN PS_EP_APPR_COMMENT B
    ON A.EP_APPR_ID = B.EP_APPR_ID
    AND A.EP_SECTION_ID = B.EP_SECTION_ID
    AND A.EP_ITEM_ID = B.EP_ITEM_ID
WHERE A.EP_APPR_ID = :appr_id
ORDER BY A.EP_SECTION_ID, A.EP_ITEM_ID
```


### User by Employee ID

```sql
SELECT OPRID, OPRDEFNDESC, EMPLID
```

```sql
SELECT 
    RU.OPRID,

FROM PSROLEUSER RU

    ON RU.ROLENAME = RD.ROLENAME
```

## Tips for PeopleSoft SQL

1. **Always use effective dating** - Most tables have EFFDT; get MAX(EFFDT) <= SYSDATE
2. **Handle EFFSEQ** - For same-day changes (especially PS_JOB), also max on EFFSEQ
3. **Use SETID for setup tables** - DEPT_TBL, LOCATION_TBL, JOBCODE_TBL require SETID joins
4. **Check HR_STATUS** - 'A' = Active, 'I' = Inactive for job records
5. **Check EFF_STATUS** - 'A' = Active, 'I' = Inactive for setup tables
6. **Use bind variables** - `:param` syntax for parameters, not string concatenation
7. **EMPL_RCD matters** - Employees can have multiple employment records (jobs)
